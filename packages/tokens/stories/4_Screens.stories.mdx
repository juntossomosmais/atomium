# Screens

All breakpoints used in our projects.

| Size   | Variable            | Device              |
| ------ | ------------------- | ------------------- |
| 360px  | `var(--screen-xs)`  | Mobile              |
| 768px  | `var(--screen-sm)`  | Tablet              |
| 992px  | `var(--screen-md)`  | Desktop             |
| 1280px | `var(--screen-lg)`  | Large Desktop       |
| 1440px | `var(--screen-xlg)` | Extra Large Desktop |
| 1920px | `var(--screen-fhd)` | Full HD             |
| 2048px | `var(--screen-2k)`  | 2K Resolution       |
| 3840px | `var(--screen-4k)`  | 4K Resolution       |

We also has a variable to define the max size of containers.

| Size   | Variable                      |
| ------ | ----------------------------- |
| 1140px | `var(--screen-max-container)` |
