## Using

Tokens could be used as CSS Variables or CSS Classes.

#### !important

We have variables to `font` and `letter spacing`, so you just need to use on `font` and `letter-spacing` property.

All the `letter-spacing` variable is the font + letter, example: `var(--title-giant-letter)`
